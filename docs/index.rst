Text Processing Toolkit Documentation
=====================================

Welcome to the Text Processing Toolkit documentation. This is a Polylith workspace for text processing applications with token-optimized architecture.

Overview
--------

The Text Processing Toolkit provides a modular and extensible framework for building text processing applications. Built using the Polylith architecture, it ensures high maintainability and code reusability.
